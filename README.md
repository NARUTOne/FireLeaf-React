## 准备reading

### install

```sh
git clone </path/to/repository>

cd repository

npm install
```

### dev-running

```sh
# 默认 3001
npm start
# 或
npm start -p [port]
# webpack-dev
npm run dev

```

### build-dist

```sh
# 规范测试校验
npm run lint
# 打包
npm run build
```
